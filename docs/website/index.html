Complete Installation Guide

Follow these 4 simple steps to get TenangDB up and running

1

Install Dependencies

Test and install required tools

make test-deps
2

Get TenangDB

Clone and build the project

git clone https://github.com/abdullahainun/tenangdb.git
cd tenangdb && make build
3

Configure

Set up your database credentials

cp configs/config.yaml /etc/tenangdb/
nano /etc/tenangdb/config.yaml
4

Run First Backup

Start your automated backup system

./tenangdb backup --config /etc/tenangdb/config.yaml
